Handicraft Shop Front-End Website

Description
This is a front-end website for a handicraft shop, created for my client as part of her university project. The website is built using HTML, CSS, and JavaScript, showcasing various types of handicrafts available for sale.

Features
The website consists of the following pages:
1.	Home: The landing page that welcomes visitors and provides an overview of the shop.
2.	Menu: A navigation page that links to different sections of the website.
3.	About Me: A page that gives background information about the shop and the creator.
4.	Contact Us: A page where visitors can find contact information and a form to reach out.
   
Additionally, the website highlights four types of handicrafts available for purchase:
1.	Wood Handicraft
2.	Palm Handicraft
3.	Coconut Handicraft
4.	Other Handicraft
   
Technologies Used
HTML: For the structure of the web pages.
CSS: For styling and layout.
JavaScript: For interactivity and dynamic content.
